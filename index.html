Creación de Cuenta en plataforma GITHUB

 

 

Introducción.

 

Estimados participantes, en las semanas quinta y sexta del curso de Introducción a la Programación, se realizará un trabajo final, para el desarrollo del mismo se hará uso de algunas plataformas colaborativas. Por tal razón en la presente semana se solicita crear una cuenta en una plataforma llamada GITHUB.

 

“GitHub es una plataforma de desarrollo colaborativo para alojar proyectos utilizando el sistema de control de versiones Git.” tomado de http://bit.ly/1kBaGDR

 

Objetivo.

  • •Crear una cuenta en la plataforma GITHUB 

 

Proceso para creación de una plataforma GITHUB

Importante Si usted ya tiene una cuenta en la plataforma GITHUB, no debe realizar la actividad.

 

 

Paso 1.

 

  • •Ingresar mediante un navegador web a la siguiente dirección: https://github.com/, Ilustración 1 

     

     
     
     

 

Paso 2.

  • •Llenar los datos de usuario necesarios para crear la cuenta en la plataforma, en base a la Ilustración 2 

 
 

1. Ingresar su nombre de usuario (campo Username)

2. Ingresar su correo electrónico (campo Email)

3. Ingrese su clave de acceso (campo Password)

4. Dar clic en el botón “Sign up for GitHub”

 

 

Paso 3.

 

  • •Dar clic en el botón “Continue”, Ilustración 3 

 

 
 

Paso 4.

  • •Dar clic en el botón “Submit”, Ilustración 4 

     

 
 

Paso 5.

  • •Si usted ha llegado a la siguiente pantalla, Ilustración 5 

 

 
 

Ha terminado la actividad.

Por favor, se recomienda recordar su nombre de usuario, correo electrónico y contraseña.  Se recuerda que la cuenta creada en la plataforma GITHUB, será usada en las siguiente temáticas.
